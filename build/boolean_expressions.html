Boolean Expressions

Relational Operators

Relational operators are used to compare two values and return a Boolean value. The six relational operators in Java are:

  • == equal to
  • != not equal to
  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to

int num2 = 7;

Logical Operators

Logical operators are used to combine Boolean expressions and return a Boolean value. The three logical operators in Java are:

  • && logical AND
  • || logical OR
  • ! logical NOT
For example, to check if a number is between 1 and 10:

int num = 5;
if (num >= 1 && num <= 10) {
⠀⠀System.out.println(num + " is between 1 and 10."); // prints this

⠀⠀System.out.println(num + " is not between 1 and 10.");

Boolean Variables

Boolean variables are variables that can only have the value true or false. They are often used to store the result of a Boolean expression. The basic syntax for declaring a Boolean variable is

boolean varName;

For example, to declare a Boolean variable and assign it the value of a Boolean expression:

int num1 = 5;
int num2 = 7;
boolean isEqual = num1 == num2; // isEqual = false
System.out.println("The numbers are equal: " + isEqual);

Short-Circuit Evaluation

Short-circuit evaluation is a feature of logical operators in Java that can improve program performance. When evaluating a logical expression using the && or || operators, Java only evaluates the second operand if necessary.
For example, if the first operand of a logical AND expression is false, then the entire expression is false regardless of the value of the second operand. In this case, Java does not evaluate the second operand. Similarly, if the first operand of a logical OR expression is true, then the entire expression is true regardless of the value of the second operand.

Equivalent Expressions with DeMorgan's Law

DeMorgan's Law is a rule in logic that describes how to negate an expression. In Java, DeMorgan's Law can be used to express a Boolean expression in an equivalent form. Some basic rules are:

  • !(a && b) equal to !a || !b
  • !(a || b) equal to !a && !b
  • !(!a) equal to a
  • !(a == b) equal to a != b
  • !(a != b) equal to a == b
  • !(x < 0) equal to x >= 0
  • !(x > 0) equal to x <= 0
  • !(x <= 0) equal to x > 0
  • !(x >= 0) equal to x < 0

int num2 = 7;
