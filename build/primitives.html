Primitive Variables

Primitive Data Types

Java has eight primitive data types, which are divided into four categories: integer, floating-point, character, and boolean.

Integer Types

Type Size (bits) Range
byte 8 -128 to 127
short 16 -32,768 to 32,767
int 32 -2,147,483,648 to 2,147,483,647
long 64 -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807

Floating-Point Types

Type Size (bits) Range Precision (decimals)
float 32 1.4E-45 to 3.4028235E38 6-7
double 64 4.9E-324 to 1.7976931348623157E308 15-16

Character Type

Type Size (bits) Range
char 16 '\u0000' (0) to '\uffff' (65,535)

Boolean Type

Type Size (bits) Range
boolean 1 true or false

Declaring Variables

To declare a primitive variable, you need to specify its data type and a name.
For example:

int age;
double weight;
char initial;
boolean isStudent;

Initializing Variables

When a variable is declared, it does not have a value. You need to assign a value to it before you can use it. You can initialize a variable when you declare it, or later in the code.
For example:

int age = 18;
double weight = 68.5;
char initial = 'J';
boolean isStudent = true;

Type Conversion

Java has strict rules for type conversion. When you assign a value of one type to a variable of another type, the value may be automatically converted to the target type.
For example:

int a = 5;
double b = a; // automatic conversion from int to double

However, some conversions may result in a loss of precision or truncation of data. You can use explicit type casting to convert a value to a different type.
For example:

double x = 3.14; int y = (int) x; // explicit casting from double to int
